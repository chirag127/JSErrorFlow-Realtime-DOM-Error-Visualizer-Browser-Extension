Icon Generator for JavaScript Error Visualizer

Instructions

This tool will generate PNG icons from the SVG icon for the JavaScript Error Visualizer extension.

  1. Click the "Generate Icons" button below.
  2. Right-click on each generated icon and select "Save Image As..."
  3. Save each icon with the appropriate name (icon16.png, icon48.png, icon128.png) in the extension/assets/icons/ directory.
